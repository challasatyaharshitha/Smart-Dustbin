# Smart Automated Trash Bin Overview

This project implements an automated trash bin that opens its lid when a user approaches, using an ultrasonic sensor for proximity detection and a servo motor for lid movement. A manual override button is also included for controlled operation.

# Technologies / Tools

Arduino  

Ultrasonic Sensor (HC-SR04)  

Servo Motor  

Push Button  

# Key Features

Proximity-based automatic lid opening  

Manual override control using button  

Real-time distance sensing  

Reduced false triggering with optimized logic  

Reliable sensor-actuator integration
